# homebrew-tap

Custom Homebrew tap for [riza](https://github.com/riza)'s tools.

## Install

```bash
brew tap riza/tap
```

## Formulae

| Formula | Description |
| ------- | ----------- |
| [cocaine](https://github.com/riza/cocaine) | Keep your machine awake. Like `caffeinate`, but with more kick. |

## Usage

```bash
# Install cocaine
brew install riza/tap/cocaine

# Or after tapping
brew tap riza/tap
brew install cocaine
```
